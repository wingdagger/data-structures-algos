# Time and Space Complexities

## Heap

|Method|Time|Space|
|-|-|-|
|Construct|O(N)|O(N)|
|Insert|O(log N)|O(1)|
|Peek|O(1)|O(1)|
|Delete (Poll)|O(log N)|O(1)|
|Get Heap Size|O(1)|O(1)|


## Heap Sort

|Method|Time|Space|
|-|-|-|
|Sort|O(N log N)|O(N)|

## Top K Problem
Return the top K elements in priority order

|Method|Time|Space|
|-|-|-|
|Construct|O(K log N + N)|O(N)|
